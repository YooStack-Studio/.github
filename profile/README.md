## 🚀 关于我们

YooStack Studio 是一个以 **创新驱动、开放共创** 为核心的技术团队，专注于探索前沿 UI 交互技术，致力于为开发者社区提供优雅、高效的开源工具与组件。

我们的理念是：**用代码让界面更有温度**。无论是液态玻璃效果、动态视觉反馈，还是跨平台组件库，我们相信每一次点击和滑动，都应该是一次愉悦的体验。

## 📦 开源项目

### [COCO-Liquid-Glass](https://github.com/YooStack-Studio/COCO-Liquid-Glass)
一个适用于 **COCO 编辑器** 的 **液态玻璃（Liquid Glass）控件**，为你的 UI 带来现代、流畅的半透明视觉效果。

- **技术栈**：JavaScript
- **开源协议**：MIT
- **特性**：轻量、易集成、高度可定制

> 🔥 这是我们的第一个开源项目，未来将有更多惊喜陆续推出！
